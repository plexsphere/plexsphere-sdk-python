# IncidentStatus

Lifecycle status of an incident: `open` while it is being worked, `resolved` once it has been resolved. The transition is one-way and permits a single resolve. 

## Enum

* `IncidentStatusOpen` (value: `'open'`)

* `IncidentStatusResolved` (value: `'resolved'`)
