# SessionKind

Kind of mediated access a Session grants: `ssh` is a mediated SSH session, `k8s` a mediated Kubernetes API session, `tcp` a generic protocol-opaque TCP tunnel. 

## Enum

* `SessionKindSsh` (value: `'ssh'`)

* `SessionKindK8s` (value: `'k8s'`)

* `SessionKindTcp` (value: `'tcp'`)
