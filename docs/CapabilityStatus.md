# CapabilityStatus

Checksum status of a Node's reported agent binary against the known-good artifact set: `match` when the reported checksum equals a known-good artifact, `drift` when it differs, and `unknown` when no known-good artifact is on record. 

## Enum

* `CapabilityStatusMatch` (value: `'match'`)

* `CapabilityStatusDrift` (value: `'drift'`)

* `CapabilityStatusUnknown` (value: `'unknown'`)
