# IdentityKind

Closed-set discriminator distinguishing human users from machine principals on the per-Domain identity-listing surface. The two values mirror the database CHECK constraint and the ReBAC schema's principal kinds. 

## Enum

* `USER` (value: `'user'`)

* `SERVICE_MINUS_IDENTITY` (value: `'service-identity'`)
