# CapacityUnit

Closed enum of the unit a capacity dimension's `used` and `target` values are expressed in. `count` is an absolute concurrent / enrolled tally; the `_per_second` units are sustained rates. 

## Enum

* `CapacityUnitCount` (value: `'count'`)

* `CapacityUnitEventsPerSecond` (value: `'events_per_second'`)

* `CapacityUnitReadsPerSecond` (value: `'reads_per_second'`)

* `CapacityUnitBytesPerSecond` (value: `'bytes_per_second'`)
