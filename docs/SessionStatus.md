# SessionStatus

Lifecycle status of a Session. `active` is the initial live state; `revoked` is set on an explicit or swept revocation; `expired` is set when the TTL elapses without a revocation. 

## Enum

* `SessionStatusActive` (value: `'active'`)

* `SessionStatusRevoked` (value: `'revoked'`)

* `SessionStatusExpired` (value: `'expired'`)
