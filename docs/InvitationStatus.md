# InvitationStatus

Closed-set lifecycle status of an Invitation. `pending` is the initial state; `accepted`, `revoked`, and `expired` are absorbing terminal states. The four values mirror the plexsphere.invitations.status CHECK constraint and the aggregate's `Status` enum. 

## Enum

* `PENDING` (value: `'pending'`)

* `ACCEPTED` (value: `'accepted'`)

* `REVOKED` (value: `'revoked'`)

* `EXPIRED` (value: `'expired'`)
