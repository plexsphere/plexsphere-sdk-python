# IntegrityViolationStatus

Triage lifecycle state of an integrity violation: `open` on intake, `acknowledged` once an operator records a reason, and `resolved` once the underlying divergence is cleared. 

## Enum

* `IntegrityViolationStatusOpen` (value: `'open'`)

* `IntegrityViolationStatusAcknowledged` (value: `'acknowledged'`)

* `IntegrityViolationStatusResolved` (value: `'resolved'`)
