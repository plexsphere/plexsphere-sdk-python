# ResourceOrigin

Closed-set discriminator on a Resource: `adopted` for a Resource that entered the platform through an out-of-band channel; `provisioned` for a Resource whose substrate is owned by the Provisioning Broker. The wire values are intentionally lowercase; the handler maps them onto the domain's `Adopted` / `Provisioned` literals. 

## Enum

* `ADOPTED` (value: `'adopted'`)

* `PROVISIONED` (value: `'provisioned'`)
