# ActionKind

Kind of remote action a dispatch invokes: `builtin` names a built-in action shipped with the Node agent; `hook` names a user-declared hook action. 

## Enum

* `ActionKindBuiltin` (value: `'builtin'`)

* `ActionKindHook` (value: `'hook'`)
